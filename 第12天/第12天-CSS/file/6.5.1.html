第1课：CSS是什么？

示例1：有鼠标悬停的链接的效果

为链接设置悬停效果十分流行。所以，我们将多看几个:hover伪类的例子。

1. 字符间距

我们在第5章学过，可以用letter-spacing属性来调整字符间距。

现在为了取得特殊效果，我们将它应用到链接上：

a:hover {
    letter-spacing: 10px;
    font-weight:bold;
    color:red;
}

2. 大写和小写

同样在第5章我们学过，可以通过text-transform属性来转换字母的大小写。

这也可用于为链接制造效果：

a:hover {
    text-transform: uppercase;
    font-weight:bold;
    color:blue;
    background-color:yellow;
}

通过上面两个例子你会发现，我们可以通过属性的组合创造出无数种效果。你可以试试创建自己的效果！
